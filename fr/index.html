TRAVAIL

QUIZ

Une application web simple où un utilisateur peut jouer en répondant à des questions sur divers sujets. L’utilisateur sélectionne une réponse dans une liste d’options afin de passer à la question suivante. Cependant, il y a un compte à rebours pour chaque question, si l’utilisateur ne sélectionne aucune réponse dans les 15 secondes, la question suivante apparaîtra jusqu’à ce que l’application donne les commentaires sur le sujet choisi par l’utilisateur

Cette application est construite à l’aide de React

PORTFOLIO

Une application Web simple qui met en valeur les compétences et les exprériences d’un individu.
